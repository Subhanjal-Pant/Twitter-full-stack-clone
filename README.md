# Twitter Full-Stack Clone

A robust, from-scratch social media application built with a focus on modern full-stack architecture, real-time interactivity, and clean engineering principles. This project replicates the core features of Twitter (X), providing a scalable foundation for a social networking platform.



## 🚀 Features

- **User Authentication:** Secure signup/login system with JWT-based session management.
- **Dynamic Feed:** Real-time tweet rendering and chronological sorting.
- **Tweet Interactions:** Full CRUD operations for tweets, including text and media support.
- **Engagement Engine:** Like and retweet functionality to drive user interaction.
- **User Profiles:** Personalized profiles showing user-specific tweets and metadata.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.

## 🛠️ Tech Stack

- **Frontend:** React.js / Next.js (for high-performance rendering and SEO).
- **Backend:** Node.js & Express (scalable server-side logic).
- **Database:** MongoDB (NoSQL flexibility for social data structures).
- **Styling:** Tailwind CSS / Styled Components (utility-first, responsive UI).
- **State Management:** Redux / React Context API.

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Subhanjal-Pant/Twitter-full-stack-clone.git](https://github.com/Subhanjal-Pant/Twitter-full-stack-clone.git)
   cd Twitter-full-stack-clone
2. ** Install Dependencies:**
   For the Backend:
    ```bash
    cd backend
    npm install
     ```
    For the Frontend:
   ```bash
   cd ../frontend
   npm install
   ```
4. ** Environment Variables:**
   Create a .env file in the backend directory and add your configurations (e.g., MONGO_URI, JWT_SECRET).
5. ** Run the Application:**
     Start the Backend:
     ```bash
     npm run dev
     ```
     Start the Frontend:
     ```bash
      npm start
     ```
